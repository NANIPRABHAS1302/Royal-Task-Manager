# 👑 Royal Task Manager

A premium **Task Management Web Application** built using **HTML5, CSS3, and Vanilla JavaScript**. Royal Task Manager helps users organize, manage, and track their daily tasks through a modern, responsive, and interactive interface.

# 📖 Project Overview

Royal Task Manager is a modern productivity application developed using pure HTML, CSS, and JavaScript. It allows users to create, organize, prioritize, and manage tasks efficiently without using any frontend frameworks.

The application features a premium glassmorphism UI, responsive design, LocalStorage support, and smooth animations to provide an excellent user experience.
